# 🎮 Simon Says Game

## 📖 About the Game

Simon Says is a classic memory game that tests a player's ability to remember and repeat an increasingly long sequence of colors. At the beginning of each level, the game highlights a random colored button. The player must carefully observe the sequence and reproduce it by clicking the buttons in the correct order. With every successful round, the sequence becomes longer, making the game progressively more challenging.

The objective is simple: **remember the sequence, repeat it correctly, and achieve the highest score possible!**

## 🎯 How to Play

1. Press **any key** to start the game.
2. Watch carefully as the game flashes a colored button.
3. Click the same colored button.
4. With each new level, one additional color is added to the sequence.
5. Repeat the entire sequence correctly to advance.
6. The game ends when you click the wrong button.
7. Try to beat your highest score!
